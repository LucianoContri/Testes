Olá, 
para rodar o código, é necessário ter o ambiente dart configurado, 
além disso ele deve ser executado pelo terminal para que seja possível captar os inputs para o teste.

Nos links abaixo estão instruções para executar arquivo dart pelo terminal
https://dartcode.org/docs/running-in-terminal/
https://dartcode.org/docs/settings/#dartcliconsole

O programa é um loop onde é recebido um entrada do tipo inteiro para imprimir o resultado
e uma entrada do tipo string para você sinalizar com "y" caso queira realizar mais testes.

Caso o input não seja inteiro para o número a ser escolhido, o programa realizará novamente outra busca por outro input.
Caso o input não seja "y" para a pergunta de continuidade, o programa se encerrará.

Espero que tenham gostado :)
Muito Grato!
